# web_fetch

获取指定 URL 的网页正文，无需打开浏览器。适合「打开这个链接并总结」「这个页面讲了什么」。

- **工具名**：`web_fetch`
- **参数**：
  - `url`：要拉取的完整 URL（必填），支持 http/https。
  - `max_chars`：返回内容最大字符数，默认 50000。
- **用法**：当用户要求总结某链接、获取某页面内容时，直接调用 `web_fetch(url="...")`，再根据返回的正文进行总结或回答。仅支持公网 URL，禁止内网地址。
