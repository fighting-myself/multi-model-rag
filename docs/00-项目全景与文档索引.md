# 项目全景与文档索引

本文档为 **AI 多模态智能问答与 RAG 指标平台**（`multi-model-rag`）的企业级文档入口，便于评审、交付与运维交接。

## 1. 项目一句话

企业级多模态 RAG 问答平台：支持多格式文档入库、向量 + 全文混合检索、流式对话与溯源、RAG 召回与六大指标评测、多模态检索、浏览器助手 / 电脑管家、Skills 与 MCP 工具扩展，配套用户认证、计费、审计与异步任务。

## 3. 代码与线上入口

| 类型 | 位置 |
|------|------|
| 后端 API 前缀 | `/api/v1`（见 `backend/app/main.py`） |
| OpenAPI 交互文档 | 服务根路径 `/docs`，ReDoc `/redoc` |
| 前端开发默认端口 | 以 `frontend/vite.config.ts` 为准（常见为 `6006`），`/api` 代理至后端 |
| 健康检查 | `GET /health` |

## 4. 维护说明

- **接口以 OpenAPI 为准**：列表文档为静态梳理，新增路由时请同步 [API接口文档.md](./API接口文档.md) 并依赖 `/docs` 校验。
- **架构/部署图**：使用 Mermaid，可在支持 Mermaid 的 Markdown 预览或 Wiki 中直接渲染；导出图片时可使用 VS Code 插件或 CI 渲染为 PNG/SVG 后贴入汇报材料。
