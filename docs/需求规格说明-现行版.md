# 需求规格说明（现行版）

> 本页描述 **当前代码库与 README 一致** 的产品能力；规划类里程碑见 [01-需求分析.md](./01-需求分析.md)。

## 1. 产品定位

**AI 多模态智能问答与 RAG 指标平台**：面向企业与团队，提供基于私有/专有文档的智能问答、检索质量与生成效果的可量化评测，并可选集成浏览器自动化、桌面自动化与 MCP 生态工具。

## 2. 功能需求（现行）

### 2.1 账户与权限

- 用户注册、登录（JWT）。
- 获取/更新当前用户信息与密码。
- 登录后访问业务页面；未登录跳转登录页。

### 2.2 文件与知识库

- 多格式上传（含 PDF、Office、图片、压缩包等，具体以后端校验为准）。
- 单文件上传、批量上传；文件列表、详情、下载、删除。
- 知识库 CRUD；向知识库添加/移除文件；分块列表查看；重索引（同步/异步）；导出等能力（以 OpenAPI 为准）。

### 2.3 智能问答（RAG）

- 基于所选知识库的多轮对话；可配置是否启用 RAG、是否 Advanced RAG（LlamaIndex 查询变换等，由环境变量控制）。
- 混合检索：向量检索 + BM25/全文，RRF 融合与 Rerank；流式输出；返回命中片段用于溯源。
- 可选联网检索与工具调用（由 `ChatService` 与配置决定）。
- 会话列表、会话详情、消息列表、删除会话。
- 对话附件上传（若接口启用）。

### 2.4 检索与多模态

- 统一检索、以文搜图、以图搜图及图片上传检索（见 `/api/v1/search/*`）。

### 2.5 评测与观测

- **召回率评测**：Benchmark 数据集管理；Recall@k、Hit@k、MRR 等指标。
- **RAG 六大指标**：准确率、召回、精准度、延迟、幻觉率、QPS 等批量/并发评测接口与默认问题集。

### 2.6 自动化与扩展

- **浏览器助手（Steward）**：基于 Playwright 的自动化任务执行接口。
- **电脑管家（Computer Steward）**：截图 + 视觉模型 + 键鼠控制类任务接口（需桌面环境）。
- **MCP**：MCP Server 配置、工具列表与调用。
- **Skills**：仓库内 `skills/<name>/SKILL.md` 供助手侧加载（产品说明见 README）。

### 2.7 运营与安全

- 仪表盘统计、计费用量与套餐、订阅与账单查询（以实际模块与配置为准）。
- 审计日志查询。
- 上传限流、审计记录等（见后端 `deps` 与 `audit_service`）。
- Bash 审批类接口（若启用相关流程）。

### 2.8 异步任务

- 长耗时任务入队（Celery）；任务状态查询。

## 3. 非功能需求（现行与目标）

| 类别 | 说明 |
|------|------|
| 性能 | 异步 FastAPI；检索与生成耗时受模型与向量库影响，流式降低首字等待；可通过水平扩展后端与独立向量库/缓存提升吞吐。 |
| 可用性 | 依赖 PostgreSQL/MySQL、Redis、向量库、对象存储、外部 LLM API 的可用性；生产建议健康检查与多副本。 |
| 安全 | HTTPS（生产）、JWT、CORS 可配置、密钥与连接串不入库；敏感配置走环境变量/密钥管理。 |
| 可扩展性 | 向量库可切换（如 Zilliz / Qdrant）；LLM 走 OpenAI 兼容接口；MCP 扩展外部工具。 |
| 可观测 | 请求 ID 中间件、审计日志；可对接 Prometheus/Grafana/ELK（见 [05-部署方案.md](./05-部署方案.md)）。 |

## 4. 用户角色

| 角色 | 能力概要 |
|------|----------|
| 普通用户 | 管理自有文件与知识库、问答、检索、评测、（可选）助手与 MCP |
| 管理员 | 在业务设计中可扩展全局配置与用户管理；具体以实际 RBAC 实现为准 |

## 5. 典型场景

- **企业知识问答**：内部文档入库后统一问答与溯源。
- **效果评估**：上线前/变更模型与分块策略后，用 Benchmark 与六大指标对比基线。
- **自动化**：通过 Steward / Computer Steward / MCP 完成浏览器或桌面侧自动化（受环境与权限限制）。

## 6. 约束与依赖

- **模型与密钥**：需配置可用的 Embedding、LLM、可选 Rerank/OCR 等。
- **基础设施**：关系库、Redis、向量库、对象存储（MinIO 或兼容 S3）。
- **可选组件**：Celery Worker、Playwright 浏览器依赖、桌面端键鼠依赖。
